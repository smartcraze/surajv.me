**Project Overview:**  
Solana Token Launchpad is a **fully functional token creation platform on Solana Devnet**. It allows users to create, mint, and receive SPL tokens directly in their wallets in a few steps, providing an intuitive interface for blockchain experimentation.

---

## Key Features

| Feature                      | Description |
|-------------------------------|------------|
| Wallet Integration            | Connects with Phantom, Solflare, and other Solana wallets |
| Token Minting                 | Mint SPL tokens on Devnet with custom metadata |
| Metadata Initialization       | Uses SPL Token Metadata for token info |
| Live Metadata Preview         | Real-time preview of token metadata in the UI |
| On-chain Execution            | Single transaction minting process |
| Custom UI Palette             | Designed with brand-specific colors and ShadCN UI |

## User Flow

```plaintext
1. Connect Solana wallet (Phantom, Solflare, etc.)
2. Fill in token details: Name, Symbol, Metadata URI, Initial Supply
3. Click "Mint Token"
4. SPL token is created on-chain and added to Devnet wallet

Use Cases
- Solana developers testing token flows
- Hackathon demos
- Projects needing custom tokens on Devnet
- Learning SPL & Web3 development
